## 概要

本リポジトリは、Unity（URP）環境におけるシェーダー実装を中心とした
グラフィックス技術ポートフォリオです。

主に以下を目的として制作しました：

- HLSLによるシェーダー実装
- PBRに基づくマテリアル表現の検証
- ライティングに対する物理的挙動の理解

---

## 開発環境

- Unity 2022.3.62f2（URP）
- HLSL
- Windows

---

## 使用アセットについて（※アセット本体は含まれていません）

本プロジェクトでは、Unity Asset Store の以下のアセットを使用することを推奨します。

- Industrial Metal Barrel PBR  
  https://assetstore.unity.com/packages/3d/props/industrial/metal-barrel-pbr-158031

本リポジトリには、上記アセットのモデル・テクスチャ等の元データは含まれていません。
アセットは Unity Asset Store の利用規約（EULA）に基づき提供されるものであり、
再配布は禁止されているためです。

### 再現手順

1. 上記アセットを Unity Asset Store から入手
2. 本プロジェクトをクローン
3. アセットをインポート
4. マテリアルにテクスチャをセット
